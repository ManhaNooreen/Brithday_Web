# 🎉 Birthday Web

A beautiful and responsive birthday greeting webpage built using **HTML** and **CSS**. It features an animated birthday cake, floating decorations, glowing candles, and a heartfelt birthday message with optional background music.

## ✨ Features

- 🎂 Animated 3-layer birthday cake
- 🕯️ 19 glowing birthday candles
- 🎈 Floating balloons and decorations
- 💖 Responsive design for desktop and mobile
- 🎨 Modern glassmorphism UI
- 🎵 Birthday music support
- 📱 Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Pacifico & Poppins)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ManhaNooreen/Brithday_Web.git
```

2. Open the project folder.

3. Double-click `index.html` or open it using **Live Server** in VS Code.
